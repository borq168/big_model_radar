#### Cloudflare Blog

- **将威胁指标实时转化为 WAF 规则**
  - **核心内容：** Cloudflare 推出了一项新功能，解决了安全团队“知道威胁但无法自动响应”的痛点。用户现在可以在 Web 应用防火墙（WAF）中，直接引用 Cloudflare Threat Events 的实时情报数据来编写规则。这意味着可以根据“威胁行为者名称（如 Tycoon 2FA）”、“目标行业/国家”和“攻击类型”等动态属性，在攻击到达前实现自动化拦截和防护。
  - **发布日期：** 2026-06-08
  - **原文链接：** [https://blog.cloudflare.com/realtime-threat-intel-waf-rules/](https://blog.cloudflare.com/realtime-threat-intel-waf-rules/)
