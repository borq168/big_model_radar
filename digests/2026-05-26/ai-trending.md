## 三、热榜观察

**1. Agent 技能/上下文优化项目爆发**
今日 Trending 中超过半数项目与“Agent 技能文件”或“Agent 上下文增强”相关（ECC、Anthropic-Cybersecurity-Skills、andrej-karpathy-skills、taste-skill、stop-slop、knowledge-work-plugins）。这些项目均在今天获得大量 star，反映出社区对 **Agent 行为调优**（输出质量、任务适配、安全边界）的强烈需求。

**2. 代码知识图谱成新热点**
Understand-Anything 和 codegraph 均以“代码知识图谱”为核心，分别面向交互式探索和 Agent 预索引。两者在功能上有重叠但定位不同：前者侧重可视化与搜索，后者侧重降低 Agent Token 消耗。该方向在今日 Top 星增榜单中占据第一和第三位，说明开发者正在寻找更高效的代码理解与 Agent 协作方式。

**3. 小众方向涌现：AI 品味与反 AI 味**
taste-skill 和 stop-slop 虽规模较小（今日 +264 / +345），但主题鲜明——不满足于让 AI 能工作，而是追求“有品味”的输出。这可能预示着 Agent 社区正在从“功能完善”向“体验调优”阶段演进。

**4. 金融 AI 基础模型入榜**
Kronos 作为一个专注于金融领域的 Foundation Model（基础模型），今日 +245 star，表明垂直领域大模型仍受关注，但其影响尚在早期，需后续观察应用落地情况。
