## Hot List Observations

**Dominant categories**: Today’s trending is heavily skewed toward **AI Agents / Workflows** and **AI Infrastructure**, with a notable focus on *agent behavior customization*. Of the 14 AI‑related trending repos, six are agent‑skill or agent‑harness projects (ECC, cybersecurity skills, taste‑skill, stop‑slop, Andrej Karpathy skills, gstack). This reflects a community maturation: developers have moved from “how do I run an agent?” to “how do I make my agent *better*?”

**Novel project types**: The emergence of **CLAUDE.md and skill‑file optimization** is the clearest new pattern. Projects like **multica‑ai/andrej‑karpathy‑skills** and **hardikpandya/stop‑slop** are single‑file configurations that promise dramatic improvements in agent output quality. Another novel type is **terminal‑optimized for agents** – **manaflow‑ai/cmux** is a full macOS terminal designed around AI coding workflows, indicating a shift in tooling to accommodate agent‑centric development.

**Knowledge graph tools** also surged: **Understand‑Anything** (+5,604) and **codegraph** (+3,161) both create queryable code graphs, reducing token consumption and tool calls. This suggests that efficient context management (a RAG‑adjacent concern) is becoming a first‑class need in agent‑heavy workflows.

**Ecosystem connections**: The lion’s share of trending skills and tools explicitly support **Claude Code, Codex, Cursor, Gemini CLI**, and similar CLI agents. This ecosystem is clearly a major driver of innovation; no single LLM release is directly cited in the data, but the agent‑tool unification trend is unmistakable.

- **Understand‑Anything** – turns any codebase into an interactive knowledge graph. Its explosive growth suggests developers are hungry for better code understanding tools that go beyond traditional search.

- **codegraph** – pre‑indexed knowledge graphs that reduce token and tool‑call overhead. A practical answer to the “context window is too small” problem for large codebases.

- **Andrej Karpathy Skills** – a single CLAUDE.md file derived from Karpathy’s observations on LLM coding pitfalls. Its rapid adoption indicates that authoritative, opinionated agent configuration files are highly valued.

- **manaflow‑ai/cmux** – a native macOS terminal built specifically for AI agents. If this pattern spreads, we may see a new category of “agent‑first” operating system tools.

- **Kronos** – a foundation model for financial markets. While smaller in stars, it represents a growing trend of domain‑specific LLMs (finance, legal, healthcare) being built and shared openly.
