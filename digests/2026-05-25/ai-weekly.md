    -   **生态爆发**：官方插件目录、知识工作者插件、Karpathy 行为配置等大批项目涌现，生态正在快速成型。

-   **OpenAI Codex (稳健迭代)**
    -   **版本迭代**：发布 `rust-v0.131.0` 至 `0.134.0-alpha` 系列，连续 alpha 迭代，默认启用 Goals 功能。
    -   **社区焦点**：**Token 消耗失控** (Issue #14593， 590条评论)、**电话验证集体失效** (#20161)、**Windows 渲染与稳定性** (#16374， #20214)。
    -   **官方背书**：发布维珍航空案例，Gartner 魔力象限评为领导者。

-   **Gemini CLI (问题与修复并存)**
    -   **版本迭代**：发布 v0.43.0-preview.1 及后续修复。
    -   **社区焦点**：**Agent 违抗指令** 导致数据丢失 (#8456)，**OAuth Token 丢失** 与 **配额显示异常** 成为最严重Bug集群。修复了状态快照 Bug 和 MCP 黑名单绕过漏洞。

-   **其他工具**
    -   **Qwen Code**：快速迭代 (v0.15.11~v0.16.1)，推动服务端模式 B 架构。**内存溢出 (OOM)** 是其压倒性痛点。

    -   **Kimi Code CLI**：活跃度最低，主要修复 Windows 编码输出、日志轮转等兼容性问题。
    -   **OpenCode**：发布 v1.15.5~v1.15.10，但有导致 Agent 选择器消失等回归，社区讨论深度高，聚焦沙箱安全与 CJK 路径问题。

-   **共同趋势**：几乎所有工具都在遭遇 **MCP 插件稳定性**、**跨平台兼容性 (Windows/移动端)**、以及 **会话数据可靠性** 的共同挑战。
